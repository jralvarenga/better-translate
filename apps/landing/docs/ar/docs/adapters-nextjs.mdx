# Next.js

`@better-translate/nextjs` هي
طبقة Next.js App Router فوق الحزمة الأساسية. تضيف ضبط توجيه اللغة،
ومساعدات الوكيل، ومساعدات الخادم ذات النطاق المحدد للطلب، وأغلفة التنقل المدركة للغة.

## متى تستخدمها

استخدم محول Next.js عندما تستخدم Next.js App Router، وتريد مسارات ذات بادئة لغوية،
أو تريد مساعدات الطلب لمكونات الخادم، أو تريد روابط وطلبات router مدركة للغة.

## routing.ts

```tsx showLineNumbers
import { defineRouting } from "@better-translate/nextjs"

export const routing = defineRouting({
  locales: ["en", "ar"] as const,
  defaultLocale: "en",
  routeTemplate: "/[lang]", // نطاق محدد: "/app/[lang]"
})
```

## request.ts

```tsx showLineNumbers
import { getRequestConfig } from "@better-translate/nextjs/server"
import { configureTranslations } from "better-translate"
import { routing } from "./routing"
import { en } from "./messages/en"
import { ar } from "./messages/ar"

export default getRequestConfig(async () => {
  const translator = await configureTranslations({ en, ar })
  return { translator }
})
```

## server.ts

```tsx showLineNumbers
import { createServerHelpers, setRequestLocale } from "@better-translate/nextjs/server"
import requestConfig from "./request"

export const {
  getLocale,
  getMessages,
  getTranslations,
  getTranslator,
} = createServerHelpers(requestConfig)

export { setRequestLocale }
```

## layout.tsx

```tsx showLineNumbers
import { hasLocale } from "@better-translate/nextjs"
import { setRequestLocale } from "@better-translate/nextjs/server"
import { notFound } from "next/navigation"
import { routing } from "./routing"

export default async function Layout({ children, params }) {
  const { lang } = await params

  if (!hasLocale(routing.locales, lang)) {
    notFound()
  }

  setRequestLocale(lang)

  return children
}
```

## navigation.ts

```tsx showLineNumbers
import { createNavigationFunctions } from "@better-translate/nextjs"
import { routing } from "./routing"

export const {
  I18nLink,
  getPathname,
  useI18nPathname,
  useI18nRouter,
} = createNavigationFunctions({ routing })
```

## الاستخدام في مكونات الخادم

```tsx showLineNumbers
import { getTranslations } from "@/lib/i18n/server"

export default async function Page({ params }) {
  await params
  const t = await getTranslations()

  return <h1>{t("hero.title")}</h1>
}
```

## التوجيه الجذري مقابل المحدود النطاق

مع `routeTemplate: "/[lang]"`،
يعيش التطبيق بالكامل تحت `/en` و`/ar` وما إلى ذلك.

مع `routeTemplate: "/app/[lang]"`،
فقط ذلك الفرع محلّى. المسارات مثل `/` و`/login` تبقى خارج التوطين.
