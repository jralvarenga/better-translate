# التثبيت

ثبّت الحزم التي تحتاجها. ابدأ بالحزمة الأساسية وأضف المحولات حسب الحاجة.

## النواة فقط

استخدم هذا عندما تريد الترجمات في TypeScript العادي، أو خادم، أو API، أو سكريبت.
لا حاجة لأي إطار عمل.

```tsx showLineNumbers
bun add better-translate
```

```tsx showLineNumbers
npm install better-translate
```

## النواة + React

استخدم هذا لتطبيقات React التي تحتاج تبديل اللغة من جانب العميل والوصول إلى
الترجمات عبر السياق.

```tsx showLineNumbers
bun add better-translate @better-translate/react
```

## النواة + Next.js

استخدم هذا لتطبيقات Next.js App Router مع المسارات ذات البادئة اللغوية، ومساعدات
مكونات الخادم، والتنقل المدرك للغة.

```tsx showLineNumbers
bun add better-translate @better-translate/nextjs
```

## النواة + Next.js + React

استخدم الحزم الثلاث عندما يحتاج تطبيق Next.js الخاص بك إلى ترجمات من جانب الخادم
وتبديل اللغة من جانب العميل.

```tsx showLineNumbers
bun add better-translate @better-translate/nextjs @better-translate/react
```

## المتطلبات

- Node.js 18+ أو Bun 1.0+
- يُنصح بـ TypeScript 5.x
- Next.js 14+ (App Router) لمحول Next.js
- React 18+ لمحول React

## الخطوات التالية

- → بعد التثبيت، اضبط إعداد الترجمة. راجع **المحولات ← النواة** لخيارات الإعداد.
- → بالنسبة لـ Next.js، تحتاج إلى خمسة ملفات: `routing.ts` و`request.ts` و`server.ts` و`navigation.ts` و`proxy.ts`. راجع **المحولات ← Next.js**.
