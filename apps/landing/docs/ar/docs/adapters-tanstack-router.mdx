# TanStack Router

`@better-translate/tanstack-router` هو
محول TanStack Router لـ Better Translate. يتبع نمط معامل اللغة الاختياري
في TanStack Router مع `/{-$locale}` ويعمل في TanStack Start
لأن Start يعمل فوق TanStack Router.

## ما يوفره

- `defineRouting(...)` للتوجيه اللغوي ذي البادئة مع معاملات اختيارية
- `createNavigationFunctions(...)` لـ `Link` و`useNavigate` و`useRouter` و`usePathname` و`useLocale` المحلّية
- `createServerHelpers(...)` و`setRequestLocale(...)` للترجمات ذات النطاق المحدد للطلب في TanStack Start

## نموذج التوجيه

افتراضياً، يفترض المحول `/{-$locale}`:

- `/about` تستخدم اللغة الافتراضية
- `/ar/about` تستخدم العربية

يمكنك تحديد نطاق توجيه اللغة لجزء فقط من تطبيقك:

```ts showLineNumbers
import { defineRouting } from "@better-translate/tanstack-router"

export const routing = defineRouting({
  locales: ["en", "ar"] as const,
  defaultLocale: "en",
  routeTemplate: "/app/{-$locale}",
})
```

هذا يبقي `/login` دون تغيير بينما يُحلّي `/app/dashboard` و`/app/ar/dashboard`.

## مساعدات الخادم لـ TanStack Start

أنشئ المترجم مرة واحدة، ثم كشف مساعدات ذات نطاق محدد للطلب:

```ts showLineNumbers
import { configureTranslations } from "better-translate"
import {
  createServerHelpers,
  getRequestConfig,
  setRequestLocale,
} from "@better-translate/tanstack-router/server"

const requestConfig = getRequestConfig(async () => ({
  translator: await configureTranslations({
    availableLocales: ["en", "ar"] as const,
    defaultLocale: "en",
    messages: { en, ar },
  }),
}))

export const { getLocale, getMessages, getTranslations, getTranslator } =
  createServerHelpers(requestConfig)

export { setRequestLocale }
```

في TanStack Start، استدعِ `setRequestLocale(params.locale ?? routing.defaultLocale)`
داخل `beforeLoad`، أو محمّل، أو نقطة دخول خادم أخرى مرتبطة بالطلب.

## أغلفة التنقل

أدخل primitives الـ router من تطبيقك:

```tsx showLineNumbers
import {
  Link,
  useLocation,
  useNavigate,
  useParams,
  useRouter,
} from "@tanstack/react-router"
import { createNavigationFunctions } from "@better-translate/tanstack-router/navigation"

import { routing } from "./routing"

export const {
  Link: I18nLink,
  getPathname,
  useLocale,
  useNavigate: useI18nNavigate,
  usePathname,
  useRouter: useI18nRouter,
} = createNavigationFunctions({
  Link,
  routing,
  useLocation,
  useNavigate,
  useParams: () =>
    useParams({
      strict: false,
      select: (params) => ({
        locale: params.locale as "en" | "ar" | undefined,
      }),
    }),
  useRouter,
})
```
