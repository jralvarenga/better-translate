# التثبيت

تعمل حزمة النواة في أي مشروع TypeScript، بغض النظر عن الإطار أو بيئة التشغيل. إذا كنت تستخدم إطاراً مثل Next.js أو React أو Astro، يمكنك إضافة المحول المناسب فوق النواة.

## النواة

```sh
npm install @better-translate/core
```

هذا كل ما تحتاجه في معظم الحالات. تعمل في Next.js وAstro وReact وBun وNode.js والسكريبتات وAPIs أو أي بيئة TypeScript.

## المحولات

المحولات هي امتدادات للنواة تضيف مساعدات خاصة بالإطار: التوجيه المدرك للغة، وcontext وhooks الخاصة بـ React، ومساعدات لكل طلب، وغيرها. لا تستبدل النواة بل تمدّها.
