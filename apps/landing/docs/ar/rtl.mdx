# RTL

يحتوي `better-translate` على دعم مدمج لـ RTL.
تُعلن أي اللغات تُكتب من اليمين إلى اليسار في الإعداد، ثم تستعلم عن الاتجاه في وقت التشغيل
باستخدام `getDirection` أو `isRtl`.

## متى تستخدمه

استخدم دعم RTL عندما يحتاج تطبيقك إلى دعم لغات مثل العربية والعبرية والفارسية والأردية.
تمنحك مساعدات الاتجاه طريقة مكتوبة النوع لتعيين سمة `dir` على عناصر HTML
بدون إدارة جدول بحث منفصل.

## إعلان الاتجاهات

أضف مفتاح `directions` إلى استدعاء `configureTranslations`. أي لغة تحذفها تكون افتراضياً `"ltr"`.

```tsx showLineNumbers
import { configureTranslations } from "better-translate"

const translator = await configureTranslations({
  availableLocales: ["en", "ar"] as const,
  defaultLocale: "en",
  messages: { en, ar },
  directions: {
    ar: "rtl",
  },
})
```

## الاستعلام عن الاتجاه

استدعِ `getDirection` للحصول على `"ltr"` أو `"rtl"` لأي لغة.
استدعِ `isRtl` للحصول على اختصار منطقي.

```tsx showLineNumbers
translator.getDirection({ locale: "ar" }) // "rtl"
translator.getDirection({ locale: "en" }) // "ltr"

translator.isRtl({ locale: "ar" }) // true
translator.isRtl({ locale: "en" }) // false
```

## تجاوز لكل استدعاء

مرّر `config.rtl` لتجاوز الاتجاه المُعدَّ لاستدعاء واحد.
هذا لا يؤثر على بحث الرسائل — يغير فقط الاتجاه الذي يتم حله.

```tsx showLineNumbers
// فرض LTR حتى للغة RTL
translator.t("greeting", {
  locale: "ar",
  config: { rtl: false },
})

// فرض RTL حتى للغة LTR
translator.t("greeting", {
  locale: "en",
  config: { rtl: true },
})
```

## التطبيق على HTML

استخدم `getDirection` عند عرض سمة `dir` على العناصر الجذرية.

```tsx showLineNumbers
// على عنصر html (مثلاً في تخطيط Next.js)
<html lang={locale} dir={translator.getDirection({ locale })}>

// على حاوية معينة
<div dir={translator.getDirection({ locale })}>
  {t("greeting")}
</div>
```

## إعداد RTL في Next.js مع Radix UI

لتطبيقات Next.js التي تستخدم Radix UI، اتبع نمط shadcn RTL:

```tsx showLineNumbers
// components/ui/direction.tsx
"use client"

import { Direction } from "radix-ui"

export function DirectionProvider({
  direction,
  children,
}: {
  direction: "ltr" | "rtl"
  children: React.ReactNode
}) {
  return (
    <Direction.DirectionProvider dir={direction}>
      {children}
    </Direction.DirectionProvider>
  )
}
```

```tsx showLineNumbers
// app/[lang]/layout.tsx
import { createLandingTranslator } from "@/lib/i18n/config"
import { DirectionProvider } from "@/components/ui/direction"

const translator = await createLandingTranslator()
const dir = translator.getDirection({ locale: lang })

return (
  <div lang={lang} dir={dir}>
    <DirectionProvider direction={dir}>
      {children}
    </DirectionProvider>
  </div>
)
```
