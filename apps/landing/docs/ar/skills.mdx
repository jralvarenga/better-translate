# المهارات

يشرح مجلد `skills/` كيفية دمج الحزم بالطريقة الصحيحة دون قراءة قاعدة الشيفرة بأكملها أولاً.

استخدمه عندما تريد إجابة بسيطة على أسئلة مثل:

- بأي حزمة أبدأ؟
- هل أحتاج React أيضاً، أم Next.js فقط؟
- كيف يتناسب MDX مع الإعداد؟
- ما هو أصغر مجموعة حزم لتطبيقي؟

## أفضل ترتيب للقراءة

1. `skills/SKILL.md` لخريطة الحزم
2. `skills/core/SKILL.md` لبدء المترجم بشكل صحيح
3. `skills/cli/SKILL.md` إذا أردت أن يُنشئ CLI ملفات locales أو يستخرج النصوص المُعلَّمة بـ `bt`
4. `skills/react/SKILL.md` إذا كانت واجهة المستخدم React أو Expo
5. `skills/nextjs/SKILL.md` إذا كان التطبيق Next.js App Router
6. `skills/md/SKILL.md` إذا كان المحتوى في `.md` أو `.mdx`
7. `skills/combined/SKILL.md` إذا كان التطبيق يمزج routing و server rendering و client hooks

## اختيار الحزم السريع

- TypeScript عادي، Node.js، APIs، أو المكتبات المشتركة: `@better-translate/core`
- React web أو Expo: `@better-translate/core` + `@better-translate/react`
- Next.js App Router: `@better-translate/core` + `@better-translate/nextjs`
- Next.js مع hooks من جانب العميل: أضف `@better-translate/react`
- TanStack Router: `@better-translate/core` + `@better-translate/tanstack-router`، وعادةً `@better-translate/react`
- Astro: `@better-translate/core` + `@better-translate/astro`
- Markdown أو MDX مُعَرَّب: أضف `@better-translate/md`
- ملفات locales مُنشأة بالذكاء الاصطناعي أو نصوص مُستخرجة تلقائياً: أضف `@better-translate/cli`

## ثبّت المهارات

يمكنك تثبيت المهارات المستضافة في المستودع مباشرة باستخدام `npx skills add`:

```sh
npx skills add https://github.com/jralvarenga/better-translate
npx skills add https://github.com/jralvarenga/better-translate --skill core
npx skills add https://github.com/jralvarenga/better-translate --skill react
npx skills add https://github.com/jralvarenga/better-translate --skill expo
npx skills add https://github.com/jralvarenga/better-translate --skill nextjs
npx skills add https://github.com/jralvarenga/better-translate --skill md
npx skills add https://github.com/jralvarenga/better-translate --skill cli
npx skills add https://github.com/jralvarenga/better-translate --skill combined
npx skills add https://github.com/jralvarenga/better-translate --skill rtl
npx skills add https://github.com/jralvarenga/better-translate --skill examples
npx skills add https://github.com/jralvarenga/better-translate --skill status
npx skills add https://github.com/jralvarenga/better-translate --skill workspace
```
