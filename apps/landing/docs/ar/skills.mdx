# المهارات

مجلد `skills/` في
المستودع هو دليل تعليمي مكتوب بلغة بسيطة. يشرح كيفية عمل المكتبة
دون الحاجة إلى قراءة الكود المصدري الكامل أولاً.

## ترتيب القراءة

إذا كنت جديداً، اقرأ ملفات المهارات بهذا الترتيب:

1. `skills/core/README.md` — محرك الترجمة: المفاتيح المكتوبة، والمعاملات، والاحتياطية، والتحميل غير المتزامن، والمساعدات العالمية.
2. `skills/react/README.md` — محول React: BetterTranslateProvider وuseTranslations().
3. `skills/expo/README.md` — Expo وReact Native باستخدام محول React الحالي.
4. `skills/nextjs/README.md` — محول Next.js: توجيه اللغة، ومساعدات الخادم، وأغلفة التنقل.
5. `skills/combined/README.md` — كيف تعمل النواة وReact وNext.js معاً في تطبيق حقيقي.
6. `skills/examples/README.md` — خريطة تطبيقات الأمثلة — انسخ التطبيق الذي يتطابق مع هندستك.
7. `skills/workspace/README.md` — كيف يتم تنظيم المستودع الموحد: الحزم مقابل التطبيقات، وسكريبتات البناء، وملفات الضبط.

## النموذج الذهني البسيط

Better Translate لها فكرة رئيسية واحدة: **اضبط الترجمات مرة واحدة، واستخدم المساعدات المكتوبة في كل مكان.**

الحزمة الأساسية تمتلك منطق الترجمة.

حزمة React تجعل هذا المنطق سهل الاستخدام في مكونات العميل.
Expo وReact Native يستخدمان حزمة React نفسها بدلاً من موائم أصلي منفصل.

حزمة Next.js تجعل هذا المنطق سهل الاستخدام مع التوجيه ومساعدات الخادم والتنقل المدرك للغة.

## أي دليل يجب أن أقرأ؟

- → تحتاج فقط الترجمات في TypeScript أو على الخادم؟ اقرأ `skills/core/README.md`
- → تبني تطبيق React؟ اقرأ `skills/react/README.md`
- → تبني باستخدام Expo أو React Native؟ اقرأ `skills/expo/README.md`
- → تبني تطبيق Next.js App Router؟ اقرأ `skills/nextjs/README.md`
- → تطبيقك له أجزاء خادم وعميل؟ اقرأ `skills/combined/README.md`
- → تتعلم بنسخ مشاريع عملية؟ اقرأ `skills/examples/README.md`
