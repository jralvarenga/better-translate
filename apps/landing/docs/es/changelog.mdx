## Versionado

Better Translate sigue el versionado semántico. Todos los paquetes en el monorepo se versionan
juntos.

- **Patch**, correcciones de errores, sin cambios en la API
- **Minor**, nuevas funciones, compatibles hacia atrás
- **Major**, cambios incompatibles
