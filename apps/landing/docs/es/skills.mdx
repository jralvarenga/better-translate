# Skills

La carpeta `skills/` explica cómo integrar los paquetes de la manera correcta sin leer todo el código base primero.

Úsala cuando quieras una respuesta simple a preguntas como:

- ¿Con qué paquete empiezo?
- ¿Necesito React también, o solo Next.js?
- ¿Cómo encaja MDX en la configuración?
- ¿Cuál es la combinación de paquetes más pequeña para mi aplicación?

## Mejor orden de lectura

1. `skills/SKILL.md` para el mapa de paquetes
2. `skills/core/SKILL.md` para iniciar el traductor correctamente
3. `skills/cli/SKILL.md` si quieres que la CLI genere archivos de locale o extraiga cadenas marcadas con `bt`
4. `skills/react/SKILL.md` si la UI es React o Expo
5. `skills/nextjs/SKILL.md` si la aplicación es Next.js App Router
6. `skills/md/SKILL.md` si el contenido está en `.md` o `.mdx`
7. `skills/combined/SKILL.md` si la aplicación mezcla enrutamiento, renderizado en servidor y hooks de cliente

## Selector rápido de paquetes

- TypeScript plano, Node.js, APIs o bibliotecas compartidas: `@better-translate/core`
- Web React o Expo: `@better-translate/core` + `@better-translate/react`
- Next.js App Router: `@better-translate/core` + `@better-translate/nextjs`
- Next.js con hooks de lado del cliente: agrega `@better-translate/react`
- TanStack Router: `@better-translate/core` + `@better-translate/tanstack-router`, y usualmente `@better-translate/react`
- Astro: `@better-translate/core` + `@better-translate/astro`
- Markdown localizado o MDX: agrega `@better-translate/md`
- Archivos de locale generados por IA o cadenas extraídas automáticamente: agrega `@better-translate/cli`

## Instala los skills

Puedes instalar los skills alojados en el repositorio directamente con `npx skills add`:

```sh
npx skills add https://github.com/jralvarenga/better-translate
npx skills add https://github.com/jralvarenga/better-translate --skill core
npx skills add https://github.com/jralvarenga/better-translate --skill react
npx skills add https://github.com/jralvarenga/better-translate --skill expo
npx skills add https://github.com/jralvarenga/better-translate --skill nextjs
npx skills add https://github.com/jralvarenga/better-translate --skill md
npx skills add https://github.com/jralvarenga/better-translate --skill cli
npx skills add https://github.com/jralvarenga/better-translate --skill combined
npx skills add https://github.com/jralvarenga/better-translate --skill rtl
npx skills add https://github.com/jralvarenga/better-translate --skill examples
npx skills add https://github.com/jralvarenga/better-translate --skill status
npx skills add https://github.com/jralvarenga/better-translate --skill workspace
```
