# TanStack Router

`@better-translate/tanstack-router` es
el adaptador de TanStack Router para Better Translate. Sigue el patrón de
param opcional `/{-$locale}` de TanStack Router y funciona en TanStack Start
porque Start corre sobre TanStack Router.

## Qué incluye

- `defineRouting(...)` para ruteo localizado con params opcionales
- `createNavigationFunctions(...)` para `Link`, `useNavigate`, `useRouter`, `usePathname` y `useLocale`
- `createServerHelpers(...)` y `setRequestLocale(...)` para traducciones por request en TanStack Start

## Modelo de ruteo

Por defecto, el adaptador asume `/{-$locale}`:

- `/about` usa el locale por defecto
- `/es/about` usa español

También puedes limitar el ruteo localizado a una parte de tu app:

```ts showLineNumbers
import { defineRouting } from "@better-translate/tanstack-router"

export const routing = defineRouting({
  locales: ["en", "es"] as const,
  defaultLocale: "en",
  routeTemplate: "/app/{-$locale}",
})
```

Eso deja `/login` intacto mientras localiza `/app/dashboard` y `/app/es/dashboard`.

## Helpers de server para TanStack Start

Crea el traductor una sola vez y luego expón helpers por request:

```ts showLineNumbers
import { configureTranslations } from "better-translate"
import {
  createServerHelpers,
  getRequestConfig,
  setRequestLocale,
} from "@better-translate/tanstack-router/server"

const requestConfig = getRequestConfig(async () => ({
  translator: await configureTranslations({
    availableLocales: ["en", "es"] as const,
    defaultLocale: "en",
    messages: { en, es },
  }),
}))

export const { getLocale, getMessages, getTranslations, getTranslator } =
  createServerHelpers(requestConfig)

export { setRequestLocale }
```

En TanStack Start, llama `setRequestLocale(params.locale ?? routing.defaultLocale)`
dentro de `beforeLoad`, un loader u otro punto server ligado al request.

## Wrappers de navegación

Inyecta las primitivas del router desde tu app:

```tsx showLineNumbers
import {
  Link,
  useLocation,
  useNavigate,
  useParams,
  useRouter,
} from "@tanstack/react-router"
import { createNavigationFunctions } from "@better-translate/tanstack-router/navigation"

import { routing } from "./routing"

export const {
  Link: I18nLink,
  getPathname,
  useLocale,
  useNavigate: useI18nNavigate,
  usePathname,
  useRouter: useI18nRouter,
} = createNavigationFunctions({
  Link,
  routing,
  useLocation,
  useNavigate,
  useParams: () =>
    useParams({
      strict: false,
      select: (params) => ({
        locale: params.locale as "en" | "es" | undefined,
      }),
    }),
  useRouter,
})
```
