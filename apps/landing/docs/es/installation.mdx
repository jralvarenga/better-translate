# Instalación

El paquete core funciona en cualquier proyecto TypeScript, sin importar el framework o entorno de ejecución. Si usas un framework como Next.js, React o Astro, puedes añadir el adaptador correspondiente encima del core.

## Core

```tsx showLineNumbers
bun add better-translate
```

```tsx showLineNumbers
npm install better-translate
```

Esto es todo lo que necesitas para la mayoría de los casos. Funciona en Next.js, Astro, React, Bun, Node.js, scripts, APIs o cualquier entorno TypeScript.

## Adaptadores

Los adaptadores son extensiones del core que añaden helpers específicos del framework: rutas con locale, context de React, helpers por request, etc. No reemplazan el core, lo extienden.

Puedes ver los adaptadores disponibles aquí:

## Requisitos

- Node.js 18+ o Bun 1.0+
- TypeScript 5.x recomendado
