# TanStack Start

`@better-translate/tanstack-start` es
un paquete scaffold para integración con TanStack Start.

> **En desarrollo** — Este adaptador aún no está completamente implementado. El paquete existe como scaffold.
> La integración con TanStack Start está en el roadmap. Por ahora, puedes usar el paquete core
> directamente con TanStack Start usando los helpers globales.

## Usando core con TanStack Start hoy

Hasta que llegue el adaptador dedicado, puedes usar el paquete core y el adaptador React
directamente en apps TanStack Start. Configura las traducciones en el punto de entrada de la app y
usa los helpers globales o el React provider en tus componentes.
