# Instalación

Instala los paquetes que necesitas. Comienza con el paquete core y agrega adaptadores según sea necesario.

## Solo core

Usa esto cuando quieras traducciones en TypeScript puro, un servidor, una API o un script.
No se requiere ningún framework.

```tsx showLineNumbers
bun add better-translate
```

```tsx showLineNumbers
npm install better-translate
```

## Core + React

Usa esto para apps React que necesiten cambio de locale en el cliente y acceso a traducciones basado en context.

```tsx showLineNumbers
bun add better-translate @better-translate/react
```

## Core + Next.js

Usa esto para apps Next.js App Router con rutas prefijadas por locale, helpers para server components
y navegación con locale.

```tsx showLineNumbers
bun add better-translate @better-translate/nextjs
```

## Core + Next.js + React

Usa los tres paquetes cuando tu app Next.js necesite traducciones en servidor y cambio de locale en cliente.

```tsx showLineNumbers
bun add better-translate @better-translate/nextjs @better-translate/react
```

## Requisitos

- Node.js 18+ o Bun 1.0+
- TypeScript 5.x recomendado
- Next.js 14+ (App Router) para el adaptador Next.js
- React 18+ para el adaptador React

## Próximos pasos

- → Una vez instalado, configura tus traducciones. Ve **Adaptadores → Core** para el formulario de opciones.
- → Para Next.js, necesitas cinco archivos: `routing.ts`, `request.ts`, `server.ts`, `navigation.ts` y `proxy.ts`. Ve **Adaptadores → Next.js**.
